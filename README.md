TWRP Device repository for ZTE Quartz zw10 (los14.1)

===========================

Codename:platy

CPU:MSM8909w

---------------
##How to compile:

1. Get the latest minimal-manifests-twrp sources
2. clone this device 
3 ./build/envsetup.sh
4. lunch lineage_platy-userdebug
5. make -jX (where X is the # of your CPU threads +1) recoveryimage
